# 📊 Análise de Cancelamento de Clientes (Churn)

Projeto de análise exploratória utilizando Python para entender padrões de cancelamento de clientes em uma empresa de telecom.

## 🎯 Objetivos
- Identificar principais motivos de cancelamento
- Avaliar impacto financeiro
- Explorar padrões comportamentais

## 🛠 Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📌 Status do projeto
Em desenvolvimento – análise exploratória concluída e aprofundamento em andamento.
